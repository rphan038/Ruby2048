This is the 2048 command line game implemented in Ruby, an object-oriented programming implementation of the game developed by  Richard Gregory Hamilton 

GitHub: @RichardGregoryHamilton
Source: https://github.com/RichardGregoryHamilton/2048-Ruby/blob/master/2048%20Ruby/lib/2048.rb

How To Run Game:


0. Install Ruby 'https://www.ruby-lang.org/en/documentation/installation/'
1. Install Bundler with command 'gem install bundler'
2. Navigate to the directory of the source code on the command-line and enter 'bundler'
3. Run the game with command 'ruby 2048.rb'
